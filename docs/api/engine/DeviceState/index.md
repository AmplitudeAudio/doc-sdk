# DeviceState

**enum class DeviceState : AmUInt8**


The device state.


    


**Opened**
:   
The device is opened, but not yet started.
         




**Started**
:   
The device is started and running.
         




**Paused**
:   
The device is paused.
         




**Stopped**
:   
The device is stopped.
         




**Closed**
:   
The device is closed.
