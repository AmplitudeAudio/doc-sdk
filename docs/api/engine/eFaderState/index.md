# eFaderState

**enum eFaderState : AmInt8**


Enumerates the list of states in a fader.


    


**eFaderState_Stopped = -1**
:   The fader is stopped


**eFaderState_Disabled = 0**
:   The fader is disabled


**eFaderState_Active = 1**
:   The fader is active and fading
