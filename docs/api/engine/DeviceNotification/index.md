# DeviceNotification

**enum class DeviceNotification : AmUInt8**


The possible device notification types.


    


**Opened**
:   
The device is opened, but not yet started.
         




**Started**
:   
The device is started.
         




**Rerouted**
:   
The device has changed.
         




**Stopped**
:   
The device is stopped.
         




**Closed**
:   
The device is closed.
